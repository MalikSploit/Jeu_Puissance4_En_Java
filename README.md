# Jeu_Puissance4_En_Java

# How to compile 
```
cd Puissance4
javac -d build src/*.java
```

# How to execute 
```
java -classpath build Main
```

# How to generate JavaDoc
```
cd JavaDoc
javadoc ../src/*.java
```

# How to open the JavaDoc
```
firefox index.html
```
Or just open the file index.html in JavaDoc directory.
